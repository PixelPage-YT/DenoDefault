### Using
> Create a default Template:
```ts
deno run --allow-all --unstable https://deno.land/x/denodefault/mod.ts
```

> Another Template (e.g. for aqua):
```ts
deno run --allow-all --unstable https://deno.land/x/denodefault/mod.ts -template aqua
```
